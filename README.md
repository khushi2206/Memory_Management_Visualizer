# Operating System Memory Management Visualizer

A React + TypeScript application to demonstrate core memory management concepts used in operating systems.

## Included Simulations

- Paging with animated frame visualization
- Page replacement algorithms:
  - FIFO
  - LRU
  - Optimal
- Segmentation layout (Code, Heap, Stack with base and limit)
- Optional comparison chart for algorithm page-fault counts

## Features

- Step-by-step transitions
- Next Step and Auto Play controls
- Page hit and page fault indicators
- Replacement highlighting
- Total page fault counter
- Animation speed control
- Clean dashboard-style UI for demos and viva presentations

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
